CLIP

Clip input tuples to a subset of attributes

SYNOPSIS

alf clip [--allbut] [OPERAND] -- ATTR_LIST

OPTIONS

--allbut                     Apply an allbut clipping?

DESCRIPTION

This operator clips tuples on attributes whose names are specified in ATTR_LIST. This is similar to the relational PROJECT operator, expect that CLIP does not remove duplicates afterwards.

Clipping may therefore lead to bags of tuples instead of sets. The result is not a valid relation unless a candidate key is preserved.

With --allbut, the operator keeps attributes in ATTR_LIST, instead of projecting them away.

EXAMPLE

# Clip suppliers on `name` and `city`
$ alf clip suppliers -- name city

    +-------+--------+
    | :name | :city  |
    +-------+--------+
    | Smith | London |
    | Jones | Paris  |
    | Blake | Paris  |
    | Clark | London |
    | Adams | Athens |
    +-------+--------+

# Clip suppliers on all other attributes
$ alf clip suppliers --allbut -- name city

    +------+---------+
    | :sid | :status |
    +------+---------+
    | S1   |      20 |
    | S2   |      10 |
    | S3   |      30 |
    | S4   |      20 |
    | S5   |      30 |
    +------+---------+
